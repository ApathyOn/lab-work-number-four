# Variant 22 - Генераторы Python

Проект демонстрирует использование функций-генераторов, многопоточности и GUI на PySide6.

## Запуск
- Локально: python app/run_gui.py
- Docker: docker-compose up --build

## Функционал
- Генератор модулей чисел
- Попарное умножение списков
- Генератор случайных паролей
- Многопоточный генератор паролей
- GUI с вкладками для каждой функции
