# Лабораторная работа №4 — Генераторы и GUI в Python

## Описание проекта

Данный проект представляет собой MVP-приложение, демонстрирующее навыки:

* Модульной структуры Python-приложений
* Объектно-ориентированного программирования (ООП) в Python
* Использования генераторов для вычислений и создания данных
* Создания графического интерфейса на PySide6
* Контейнеризации приложения с помощью Docker

Приложение выполняет три задания:

1. **Модули чисел**
   Пользователь вводит два целых числа (a) и (b) ((a < b)), программа возвращает модули чисел в диапазоне [a, b] и выводит первые четыре значения.

2. **Генератор паролей**
   Генерируются случайные пароли длиной 12 символов из набора букв, цифр и специальных символов. Выводятся первые пять паролей.

3. **Попарное умножение списков**
   Пользователь вводит два списка целых чисел, программа перемножает их элементы попарно и выводит первые три значения.

## Установка и запуск

### Вариант 1 — без Docker

```bash
git clone <репозиторий>
cd variant22_project
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app/run_gui.py
```



После запуска откроется GUI-приложение, где можно выполнять три задания через кнопки.

---

### Вариант 2 — через Docker

```bash
sudo docker build -t gui-generator-app .
sudo docker run -it --rm -e DISPLAY=$DISPLAY -v /tmp/.X11-unix:/tmp/.X11-unix gui-generator-app
```


**Примечание:** Для корректной работы GUI в Docker может потребоваться настройка X11.

## Тестирование

Используется `pytest`. Проверяются:

* Корректность генераторов (`abs_range`, `pairwise_products`, `password_generator`)
* Правильность генерации первых N элементов
* Корректная работа GUI с вводом данных

Запуск тестов:

```bash
pytest -v
```
